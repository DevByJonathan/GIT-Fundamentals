GIT Commands

Initializing a GIT repo.

  1. To initiate a repo - git init

Creating a gitignore file.

  1. First install this extension codezombiech.gitignore on your Visual Studio Code
  2. Search for Add gitignore in your Command Palette and chose

Adding files to a repo.

  1. To add a file to a repo - git add [filename.extension]
  2. To add all the files in the opened directory to the repo- git add .

Commiting files to a repo.

  1. To commit files to a repo - git commit -m ["write the message inside the double quotes and delete the brackets"]
  2. To commit a particular file after making changes in it - git commit -am [file.extension]

Basics about Branch

  • To check the branches you have - git branch
    PS."master*" is the default branch.
  • To create a new branch -
    1. git branch [branchName]
    2. git checkout -b [branchName]
      PS.This command is used to create a new branch and also switch to it after creating it
  • To switch to another branch that's already created - git checkout [branchName]
  • To delete a branch if it has already been pushed and merged with the remote branch - git branch -d [branchName]
  • To delete a branch even if it hasn't been pushed or merged yet - git branch -D [branchName]
